# Dobrodošli na UX Projekt

Ovaj projekt izrađen je kao dio zadatka za demonstraciju osnovnih GitHub funkcionalnosti, uključujući:

- rad s Markdown datotekama  
- dodavanje slika iz besplatnih izvora  
- umetanje kodnih blokova  
- kreiranje LICENSE i CHANGELOG datoteka  
- aktiviranje GitHub Pages-a  

Cilj projekta je prikazati kako se može strukturirati jednostavna dokumentacija koristeći GitHub Pages kao platformu za objavu.

## Sadržaj repozitorija

- **README.md** – osnovni opis projekta  
- **LICENSE.md** – MIT licenca  
- **CHANGELOG.md** – popis verzija i promjena  
- **index.md** – početna stranica GitHub Pages-a  

## O projektu

Projekt se fokusira na UX elemente, osnovnu dokumentaciju i organizaciju repozitorija.  
Služi kao primjer kako izraditi jasan, uredan i funkcionalan GitHub projekt koji se može javno prikazati putem GitHub Pages-a.
